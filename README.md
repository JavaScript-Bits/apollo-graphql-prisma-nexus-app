# apollo-graphql-prisma-nexus-app
A sample Express app to demonstrate using Apollo GraphQL with Nexus and Prisma with TS

## Set Up
### npm
`npm install`

### Set up `env`
`cp .env.example .env`

Edit database envs, etc

### Set up prisma
`npm run migrate`

### Run Server
`npm run dev`


### Build
`npm run build`
